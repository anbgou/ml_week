# Plant Diseases Classification

Цей проєкт присвячений задачі **бінарної класифікації зображень листя** на два класи: **healthy** (здорове) та **sick** (хворе).  
Основна ідея — перевірити, чи можна **покращити якість моделі** за рахунок великого пулу **нерозмічених даних** через напівконтрольоване навчання (**pseudo-labeling**).

> Основний файл для перегляду результатів, пояснень і коду:  
> **`Plants_PseudoLabeling_Full_Report.ipynb`**

---

## Що зроблено в проєкті

У ноутбуці послідовно реалізовано повний пайплайн дослідження:

**Підготовка та правила експерименту**
   - фіксація `seed` для відтворюваності
   - правило недоторканості тестової вибірки (test не використовується для навчання / псевдорозмітки / підбору порога)

**Завантаження та аналіз даних**
   - перевірка структури папок `train/val/test` та `unlabeled/images`
   - підрахунок кількості зображень по класах
   - обчислення пропорцій класів (дисбаланс)
   - візуалізації та приклади зображень


 **Побудова baseline моделі (Model v1)**
   - **ResNet-18 (ImageNet pretrained)**
   - transfer learning: warmup (fc) → fine-tuning (layer4 + fc)
   - оцінка метрик на validation та test
   - підбір оптимального **threshold** на validation для максимуму **F1-score**
   - графіки навчання (loss curves), ROC/PR curves, confusion matrix

**Pseudo-labeling нерозмічених даних**
   - інференс baseline моделі на 5000 unlabeled зображень
   - відбір лише впевнених прогнозів за порогом confidence (наприклад ≥ 0.90)
   - балансування псевдоміток за класами
   - формування папки `pseudo/train/...` та збереження CSV із псевдорозміткою
   - візуалізації: гістограма confidence

**Навчання розширеної моделі (Model v2)**
   - об’єднання labeled + pseudo-labeled у розширений train
   - повторне навчання з контролем впливу псевдоміток (loss weight)
   - early stopping, threshold tuning на validation
   - фінальна оцінка на test та порівняння v1 vs v2

**CLIP zero-shot baseline**
   - запуск CLIP у zero-shot режимі з prompt ensembling
   - порівняння з CNN-підходом
